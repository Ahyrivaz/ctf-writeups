# CTF Writeups

A curated collection of practical CTF and lab notes focused on real-world offensive security skills.

This repository gathers walkthrough-style writeups from platforms like **TryHackMe** and **Hack The Box**

## Why This Repo

The goal is to keep a clean, centralized knowledge base of solved labs and techniques that can be quickly revisited for:

- Exam prep
- Red team practice
- Skill reinforcement
- Building personal methodology

## How To Use

1. Pick a topic (e.g., AD, PrivEsc, WiFi).
2. Open the relevant PDF writeup.
3. Reproduce the steps in a legal lab environment.
4. Compare your method with the documented approach.
5. Take your own notes and iterate.

## Ethics & Legal Notice

All techniques and notes in this repository are intended **for educational purposes only**.
Only test systems you own or have explicit permission to assess.

If you are learning offensive security, this repo is designed to be a practical, repeatable reference library rather than just a file dump.
